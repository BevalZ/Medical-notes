## 如何做好病历书写

### 要求

- 客观
- 真是
- 准确
- 及时
- 完整
- 规范

### <font color="red">**时限控制项目**</font>

- [ ] 6小时内：抢救病历
- [ ] 8小时内：首次病程记录
- [ ] 24小时内：住院记录、手术记录、转入（出）记录、交（接）班记录、出院记录
- [ ] 48小时：首次主治医师查房记录
- [ ] 72小时：首次（副）主任医师查房记录<sub>**未在上述时限内完成病案书写的，视为乙级病历**</sub>
- [ ] 30天内：完成一次小结
- [ ] 病危：每天至少1次（病情变化随时记录）
- [ ]  病危：每2天至少1次（病情变化随时记录）
- [ ] 病情稳重：每3天至少1次

### 住院记录
#### 1.主诉

**症状（体征）**+持续时间<sub>**如果反复就诊或者本次就诊目的明确--治疗，主诉可写入本次治疗目的**</sub>
**不超过20字**

#### 2.现病史

-现病史与主诉相符
- 起病时间与诱因
- 主要症状、体征的**部位、时间、性质、成都**，伴随症状与体征描述
- 有鉴别意义的阴性症状、体征
- 疾病演变、入院前诊治经过及效果（**注意写清入院前治疗检查医院及时间**）

<sub><font color="red">**注：本次住院需要治疗的其他疾病，另起一行书写**</sub> </font>

#### 3.既往史

- 既往各系统的情况
- 手术、外伤史，重要的传染病病史、输血史、过敏史（药物、食物、其他）

#### 4.个人史

- 个人生活习惯、嗜好(吸烟:吸烟史xx年，xx支年[吸烟年数*每天吸烟数量],)、职业接触史、不良性生活史（写否认而不写无）
- 婚育史（时长、生育史、子女）、月经史

- 遗传疾病（病史询问不少于三代）

### 首次病程

#### 1.病例特点：

归纳提炼突出重点（病史、体征、辅助检查三个方面）

#### 2.诊断与鉴别诊断

- 紧扣刘病例特点
- 分析过程，阐明诊断依据和鉴别诊断
- 必要时进行重难点分析讨论

**鉴别诊断无须注意鉴别，但要对主要诊断详细书写**

#### 3.诊疗计划

具体、针对性

### 围手术期记录

#### 术前小结

**手术者术前查看患者相关情况需要记录**

#### 术前讨论记录

**重点：**

- 术式
- 术中可能出现的问题及其应对措施

#### 手术记录

**重点:**

- 手术一般情况
- 手术经过
-特殊情况及其处理（术中植入物条形码应该粘贴到病历中）

**注意：术后24小时内完成，应该由手术者或者是一助书写（但是都需要手术者签字）**

- 术后当天病情记录由参加手术人员书写
- 术后联系记录**3天病程（不包含手术当日）**
- 术后3天内上级医师或术者查看病人记录

### 抢救记录

抢救结束后**6小时内**据实补记，加以注明

内容：病情变化情况、抢救时间及措施、参加抢救的医务人员姓名及专业技术职称

**抢救时间具体到分钟**

**必须由参与抢救的人员进行书写**

### 死亡病例讨论记录

患者**死亡一周内**,主任或具有副主任医师以上专业技术职称资格的医师主持

内容包括讨论日期、主持人及参加人员姓名、专业技术职务、具体讨论意见及主持人小结意见、记录者的签名等

### 出院记录
是指经治医师对患者此次住院期间诊疗情况的总结,应当在患者出院后**24小时内完成**。

内容主要包括入院日期、出院日期、**入院情况、入院诊断、诊疗经过、出院诊断、出院情况、出院医嘱**、医师签名

### 其他

- [ ] 对病情变化要有分析及相应的处理意见
- [ ] 对检查结果异常的分析及相应的处理意见
- [ ] 病程记录要反映会诊意见及执行情况
- [ ] 出院前上级医师同意出院记录及上级医师签字
- [ ] 死亡记录中死亡时间与医嘱、体温单时间要一致
- [ ] 有证据证明病历记录为拷贝—-乙级

### 主要诊断


主要诊断：经研究确定的导致患者本次住院就医主要原因的疾病(或健康状况)。

疼痛科:主要是为了治疗或控制疼痛；应该选择”疼痛的相关诊断“，而非原发病诊断

患者就诊的目的是为了做手术治疗引起疼痛的原发疾病(椎间盘突出),则该原发疾病的应该被作为主要诊断。

主要诊断一般应该是:
- 消耗医疗资源最多
- 对患者健康危害最大
- 住院时间最长
- 多发伤，以明确的最严重损伤(由深至浅）的疾病诊断为主要诊断<br>
&emsp;&emsp;主要诊断：脾破裂<br>
&emsp;&emsp;其他诊断：小肠破裂<br>
&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;前臂骨折<br>
- 主要并发症或伴随疾病

一般情况下,有手术治疗的患者的主要诊断要与主要手术治疗的疾病相一致。
